Corollary 7.5.3.

Let \(\chi\) be a nonprincipal Dirichlet character modulo \(q\text{.}\) Then for all \(x \geq 2\) we have
\begin{align} \sum_{n \leq x} \frac{\chi(n)}{n} \amp = L(1,\chi) + O\left(\frac{1}{x}\right) \tag{7.5.16}\\ \sum_{n \leq x} \frac{\chi(n)\log n}{n} \amp = \sum_{n=1}^{\infty} \frac{\chi(n) \log n}{n} + O\left(\frac{\log x}{x}\right) \tag{7.5.17}\\ \sum_{n \leq x} \frac{\chi(n)}{\sqrt{n}} \amp = L\left(\frac{1}{2},\chi\right) + O\left(\frac{1}{\sqrt{x}}\right) \tag{7.5.18} \end{align}

Proof.

The previous theorem with functions \(\frac{1}{x}\text{,}\) \(\frac{\log x}{x}\text{,}\) and \(\frac{1}{\sqrt{x}}\) respectively.
